Detailed Description

A neural network interface.

This class is a generic interface for neural net classes.

Source: Russell S. and Norvig P. 1995. "Neural Networks" Artificial Intelligence: A Modern Approach. Prentice Hall, Upper Saddle River, N.J.

Note
This is an abstract base class.
