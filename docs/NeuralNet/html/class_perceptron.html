Detailed Description

The classic Perceptron neural net.

The classic Perceptron neural net is a single-layer feed-forward network.

Source: Russell S. and Norvig P. 1995. "Perceptrons" Artificial Intelligence: A Modern Approach. Prentice Hall, Upper Saddle River, N.J.
