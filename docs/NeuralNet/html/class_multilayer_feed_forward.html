Detailed Description

A multilayer feed-forward neural net with back-propagation.

Source: Russell S. and Norvig P. 1995. "Multilayer Feed-Forward Networks" Artificial Intelligence: A Modern Approach. Prentice Hall, Upper Saddle River, N.J.

nHiddenThe number of hidden units.
nOutputsThe number of outputs.
aWeightsThe weights for each input of each unit. There must be ( nInputs + nOutputs ) * nHidden weights. The first nInputs * nHidden weights are the input weights for the hidden units. The next nOutputs * nHidden weights (the rest) are the input weights for the output units.
