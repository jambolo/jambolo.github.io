Sets the current position relative to a specific point in the file. Returns the new position.

Parameters
offNumber of uncompressed bytes to move the pointer
wayLocation to start seek. std::ios_base::end is not supported. Valid values are:
  • std::ios_base::beg
  • std::ios_base::cur
openmodeIgnored (both in and out pointers are moved)
Note
There are restrictions imposed by zlib:
  1. std::ios_base::end is not supported as a start location
  2. Only forward seeks are allowed in output buffers
