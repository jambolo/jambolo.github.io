Detailed Description

A memory stream buffer that compresses and decompresses data using zlib.

Constructor & Destructor Documentation

◆ zmembuf() [1/3]

zmembuf::zmembuf ( std::ios_base::openmode  mode)
explicit
Parameters
modeDirection of the stream
  • std::ios_base::in signifies an input buffer. Data is decompressed as it is streamed from of this buffer. You must initialize the contents of the buffer before streaming.
  • std::ios_base::out signifies an ouput buffer. Data is compressed as it is streamed to this buffer. The buffer will grow as data is streamed to it.

Parameters
offNumber of uncompressed bytes to move the pointer

  • std::ios_base::beg
  • std::ios_base::cur
whichIgnored (both in and out pointers are moved)
Note

  1. std::ios_base::end is not supported as a start location
  2. Only forward seeks are allowed in output buffers
