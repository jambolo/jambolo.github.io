Detailed Description

Buffered object.

Derive from this class to implement functions required by a Buffer. The Buffer reads and writes data to/from the buffered object through this interface.

Parameters
handleHandle provided to the Buffer.
pBufferLocation to put the data.
nNumber of blocks to read.
Returns
Number of bytes actually read – must be a multiple of the block size
Note
handle's "current location" is expected to move to the next byte following the data that was copied to the buffer.
n is always a multiple of the block size.
pBuffer is always aligned to the buffer alignment.
