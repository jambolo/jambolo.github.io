pathfinding module¶

pathfinding.aStar(graph, start, goal, h, neighborsOf, IMPASSABLE)[source]¶

Returns the lowest cost path in a graph from the start to the goal.

If the returned path is empty, then there is no path.

Parameters:

  • graph (list) – List of the nodes in the graph.

  • start (int) – Index of the starting node in the graph.

  • goal (int) – Index of the goal node in the graph.

  • h (Callable) – Function h(n) -> cost, estimates cost from node n to the goal.

  • neighborsOf (Callable) – Function neighborsOf(n) -> [(n, cost), …], returns neighbors of node n.

  • IMPASSABLE (Any) – Value greater than the longest possible path.

Returns:

The lowest cost path as a list of node indices.

Return type:

list

pathfinding.dynamicAStar(start, goal, h, neighborsOf)[source]¶

Returns the lowest cost path in a dynamic graph from the start to the goal.

If the returned path is empty, then there is no path.

Parameters:

  • start (int) – Non-negative number representing the starting node.

  • goal (int) – Non-negative number representing the goal node.

  • h (Callable) – Function h(t, n) -> cost, estimates cost from node n to the goal at time t.

  • neighborsOf (Callable) – Function neighborsOf(t, n) -> [(n, cost), …], returns neighbors at time t.

Returns:

The lowest cost path as a list of node indices.

Return type:

list
