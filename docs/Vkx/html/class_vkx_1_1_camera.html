void Vkx::Camera::setAspectRatio ( float  w,
float  h

Sets the view window aspect ratio.

Parameters
wWidth of the image on the screen.
hHeight of the image on the screen.
Note
The units of w and h are not important. w and h are only used to compute the aspect ratio. If the pixels are square then w and h can simply be the width and height in pixels, otherwise they should be the actual width and height of the image on the screen.
