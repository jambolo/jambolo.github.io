The function returns when the queue is idle.

The function parameter should add commands as normal to the command buffer parameter, like this:

commands.copyBuffer(src, dst, vk::BufferCopy(0, 0, size));
});

◆ StripGrid() [1/2]

int Vkx::StripGrid ( int  w,
int  h,
uint16_t *  pData 
)

This function generates an array of vertex indexes for stripping a w by h grid of vertexes.

The strip is created like this:

  • For every two rows in the grid:
    • Strip a row from left to right
    • Use degenerate triangles to continue the strip down to the end of the next row.
    • Continue stripping the next row from right to left.
    • Use degenerate triangles to continue the strip down to the beginning of the next row.

The vertexes are stripped for a CW front-face.

Parameters
wWidth of the grid (note: this is the number of vertexes – the number of quads is w - 1)
hHeight of the grid (note: this is the number of vertexes – the number of quads is h - 1)
pDataWhere to place the indexes. Index values range 0 - 65535. The buffer must be big enough to hold at least ( h - 1 ) * ( 2 * w + 2 ) - 2 indexes.
Returns
Number of vertex indexes generated: ( h - 1 ) * ( 2 * w + 2 ) - 2
Warning
w * h must be less than 65536
