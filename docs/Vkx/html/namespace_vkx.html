This function creates a one-time command buffer and executes it.

Creates and executes a one-time command buffer.

The function returns when the queue is idle.

The function parameter should add commands as normal to the command buffer parameter, like this:

executeOnceSynced(device, commandPool, queue, [src, dst, size] (vk::CommandBuffer commands) {
commands.copyBuffer(src, dst, vk::BufferCopy(0, 0, size));
});
Parameters
deviceThe command buffer is allocated from this device
commandPoolCommands are allocated from this pool
queueThe command buffer is executed in this queue
commandsAdds commands to the specified command buffer
Note
commandPool should specify a pool optimized for transient command buffers
