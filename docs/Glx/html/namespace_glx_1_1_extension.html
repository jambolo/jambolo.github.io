bool Glx::Extension::IsSupported ( char const * pExtensionName)

This function searches the OpenGL extensions string for a matching extension name to determine if the extension is supported.

Returns true if the extension is supported.

Parameters
pExtensionNameExtension name to search for. The extension name string must not contain any spaces (including at the beginning or the end). Empty names are reported as unsupported.
