void Glx::DirectionUpCamera::Point ( Vector3 const & direction)

Sets the camera direction.

This function "points" the camera.

The camera's new up vector is set such that it is coplanar with the direction vector and the old up vector.

Parameters
directionDirection. The vector must be normalized();

◆ Roll()

void Glx::DirectionUpCamera::Roll ( Vector3 const & up)

Rolls the camera.

This function "rolls" the camera – the direction is not affected.

The camera's new up vector is not simply set to the specified value. Instead, it is set such that it is coplanar with the direction vector and up.

Parameters
upNew up vector. The vector must be normalized();
