This constructs an uninitialized texture.

Parameters
w,hTexture width and height. Must be a power of 2.
wrapWrap mode. The value can be any valid value for the GL_TEXTURE_WRAP_S texture parameter.
minFilteringFiltering mode used when texels are "minified". The value can be any valid value for the GL_TEXTURE_MIN_FILTER texture parameter.
magFilteringFiltering mode used when texels are "magnified". The value can be any valid value for the GL_TEXTURE_MAG_FILTER texture parameter.
idTexture id. If 0 is specified, then an id is generated.
Note
This constructor will throw ConstructorFailedException if id is 0 and there are no more texture ids available.

Constructor.

This constructs a texture from an image (if specified).

The texture has no border. The internal format is determined automatically by the image format.

Parameters
w,hTexture width and height. Must be a power of 2.
pImageImage data
formatFormat of the image data - components. The value can be any valid format (not "internal" format) used by glTexImage().
typeFormat of the image data - value type. The value can be any valid data type used by glTexImage().
wrapWrap mode. The value can be any valid value for the GL_TEXTURE_WRAP_S texture parameter.
minFilteringFiltering mode used when texels are "minified". The value can be any valid value for the GL_TEXTURE_MIN_FILTER texture parameter.
magFilteringFiltering mode used when texels are "magnified". The value can be any valid value for the GL_TEXTURE_MAG_FILTER texture parameter.
