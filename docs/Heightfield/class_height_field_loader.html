Creates a HeightField from a TGA file.

This function creates a HeightField using height data loaded from a TGA file.

The size of the height field is determined by the size of the image. The format of the image must be one byte per pixel (IMAGE_COLORMAPPED or IMAGE_GREYSCALE). The origin of the heightfield is the origin of the image (as specified by the pixel order). The values in the image are multiplied by zScale / 255 to get the actual heights.

Parameters
sFileNameThe name of the TGA file containing the height data.
zScaleScale factor for height data
Returns
The address of the heightfield or 0 if the file could not be loaded
Note
As a result of scaling, the heights will be in the range of 0 - zScale, inclusive.

This function writes a HeightField to a TGA file.

Only the size of the height field and the Z values are saved. The value of each pixel is limited to the range [0, 255].
