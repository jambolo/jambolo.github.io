int sizeJ,
float zScale,
unsigned __int8 const * pData )

Constructor.

This constructor creates a heightfield given an array size, scale, and an array of heights.

The values in the height data are multiplied by zScale / 255 to get the actual heights.

Parameters
sizeISize of the heightfield along the I axis.
sizeJSize of the heightfield along the J axis.
zScaleScale factor for height data
pDataHeight data to be copied. The data should be in this order: pData[i][j].
Exceptions
bad_allocUnable to allocate an array of vertexes.
Note
As a result of scaling, the stored heights will be in the range of 0 - zScale, inclusive.

float HeightField::GetInterpolatedZ ( float j,
float i,
int step = 1 ) const

Returns the interpolated Z at [ j, i ].

Parameters
jj index (can be a non-integer, but must be less than the width of the heightmap)
ii index (can be a non-integer, but must be less than the height of the heightmap)
stepwidth and height of the quad to interpolate

The function assumes that every quad in the grid is triangulated like this:

@begincode (a,b+step) (a+step,b+step) +----------—+ | / | | *<--------— (j,i) | / | | / | | / *<--— or (j,i) | / | +----------—+ (a,b) (a+step,b)

Where a and b are integers and multiples of step
